Cada cuenta es administrador de su propio espacio de trabajo.
Cuando una cuenta crea un nuevo proyecto, puede enviar solicitudes a otros usuarios para que participen.
Cuando los usuarios aceptan la invitacion se les deben asignar roles especificos.
El administrador de proyecto puede designar esos roles (incluso designar a otros usuarios como administradores)

Unicamente el administrador puede crear un nuevo proyecto.
Al crarse un nuevo proyecto el sistema creara un panel de kanban por defecto (que puede modificarse)
